Repository to practice proper git usage, using claude code, and to learn PyTorch through examples on my way to train neural networks to control power converters.

Git Objectives:
 - Maintain clean working tree

Claude Code Objectives:
 - Understand Claude.md and memory files
 - Learn to use agents

PyTorch Objectives:
 - Basic PyTorch skills (Tensors, gradient descent, loss function, activation functions)
 - NN types and their PyTorch implementation
 - Using CUDA to train models
 - Generating training data

USAGE:
Create a virtual python environment. Then run ```pip install requirements.txt```
